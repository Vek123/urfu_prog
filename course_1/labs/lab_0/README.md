# Task 1

Игра-лабиринт

## Level 1

Лабиринт с 1 монстром, которого нельзя косаться. Грааль приведёт к следующему уровню.

## Level 2

Лабиринт с 2 монстрами, шипы появляются переодически, их тоже нельзя косаться.

## Level 3

Лабиринт с 2 монстрами и стенами-врагами. Коснёшься стены - начинать сначала :).
